# Foobar

 Metin şifreleme işlemlerini kolayca yapabileceğiniz bir kobra kütüphanesi. Bu paket ile verilerinizi kolayca
şifreleyebilirsiniz.

## Installation



```bash
pip install verdata
```

## Usage

```
import verdata
verdata.cal()


```
